455 /*
456  * Write text to display.
457  *
458  * Rotation of the displayed text depends on 'font_rotate' variable (0~360)
459  * if 'font_transparent' variable is set to 1, no background pixels will be printed

461  * If the text does not fit the screen width it will be clipped (if text_wrap=0),

467  *
468  * Params:
469  * st: pointer to null terminated string to be printed
470  * x: horizontal position of the upper left point in pixels
471  * Special values can be entered:
472  * CENTER, centers the text
473  * RIGHT, right justifies the text
474  * LASTX, continues from last X position; offset can be used: LASTX+n
475  * y: vertical position of the upper left point in pixels
476  * Special values can be entered:
477  * CENTER, centers the text
478  * BOTTOM, bottom justifies the text
479  * LASTY, continues from last Y position; offset can be used: LASTY+n

481  */
